MINOR PROJECT 

Use **node server.js** to run the application on local host and make sure express is installed.
To install express type - npm install express.
Gemini API key is hard coded upload your own to run the Shaodow.

Shadow AI is an AI chatbot that does not act as an assistant but rather as a rival. You input your day to day activities Shadow processes it and give you its version of the day where it has same choices as the user and the same situations but chose the best options.
